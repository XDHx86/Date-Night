## Project Purpose

Datenight guides users through planning a perfect date night by:
1. Asking a yes/no question to start
2. Picking a date
3. Selecting a time
4. Choosing a movie
5. Reviewing the selections
6. Displaying a celebratory summary

The app features engaging UI animations, a persistent state store, and a movie search backed by a curated list (with optional TMDB integration).
